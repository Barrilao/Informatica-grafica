Teclas:

Q -> exit
1 -> modo puntos
2 -> modo líneas
3 -> modo sólido
4 -> modo ajedrez
5 -> modo suavizado plano
6 -> modo suavizado de Gouraud
P -> pirámide
C -> cubo
O -> objeto ply (beethoven)
R -> objeto por rotación
E -> esfera
V -> cono
D -> cilindro
I -> objeto ply por revolución (en este caso he hecho un ovni)
T -> tanque
F1/F2 -> arriba/abajo cañón del tanque
F3/F4 -> izquierda/derecha parte de arriba del tanque
A -> ovni (práctica 3). El objeto es grande, se debe alejar la cámara
F5/F6 -> cierra/abre las patas, hasta un límite
F7/F8 -> traslada y rota la vaca, hasta un límite
F9/F10 -> rota el ovni hacia un lado y el haz de luz hacia otro, en el caso de pulsar la otra tecla hará estos movimientos inversos.
F11/F12 -> avanza/retrocede el ovni persiguiendo a la vaca
X -> activa la animación, que en este caso consiste en que cierra las patas, gira el ovni de izquierda a derecha, el haz de luz de derecha
    a izquierda mientras absorve y devuelve la vaca a su lugar.
Z -> para la animación
N -> baja la velocidad de la animación, si llega a 0 parará la animación
M -> aumenta la velocidad de la animación. En el caso de estar parada inicia la animación a la velocidad mínima.
Ins(en el teclado numérico) -> rotar iluminación
Inicio(en el teclado numérico) -> subir iluminación
Fin (en el teclado numérico) -> bajar iluminación
